# Kuan Chen - Interactive Greeting Page

A sleek, modern, and highly interactive personal greeting page built with vanilla web technologies. Featuring a responsive glassmorphic card, live clock display, smooth dark/light mode transition, and responsive layout.

## ✨ Features

- **Dynamic Interactive Card**: Features a premium 3D tilt/parallax hover effect responsive to mouse cursor coordinates.
- **Ambient Glow Backgrounds**: Implements custom floating animated gradient orbs and a subtle masked grid overlay.
- **Real-time Clock & Date**: Displays an elegant live-updating digital clock (12-hour format) with blinking separators and local date formatting.
- **Seamless Theme Toggling**: Fluid transitions between curated dark and light color systems, persisting user preferences using `localStorage`.
- **Pure Vanilla Tech Stack**: Zero external UI dependencies, built entirely using semantic HTML5, modern CSS3 variables, and vanilla ES6 JavaScript.

## 🛠️ Technology Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom properties/variables, flexbox/grid layout, 3D perspective transforms, keyframe animations)
- **Logic**: Vanilla ES6 JavaScript (DOM manipulation, MouseMove event tracking, Web Storage API)
- **Typography**: Google Fonts (Outfit & JetBrains Mono)

## 🚀 Quick Start / How to Run

Since the project has no build steps or dependencies, you can run it directly:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/bigkuan-chen/L2-DIC1-github.git
   ```
2. **Open the project**:
   - Double-click the `index.html` file to open it in any web browser, or:
   - Run a local static server:
     ```bash
     # Using Python (built-in)
     python -m http.server 8000
     
     # Or using Node.js (serve)
     npx serve .
     ```
3. Visit `http://localhost:8000` (or the port provided by your server) to interact with the page.
